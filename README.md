## 🚀 Installation

> ***Follow these instructions to set up WA-Multi-Downloader on Termux (Android), Windows, or Linux.***

---

### 🧩 Prerequisites

Before installing, make sure your system has:

| Tool | Check Command | Minimum Version |
|:---|:---|:---|
| **Node.js** | `node -v` | `>= 20` |
| **npm** | `npm -v` | *(comes with Node.js)* |
| **Git** | `git --version` | Any recent version |

> ***Also make sure you have a stable internet connection.***

---

### 📱 Termux (Android)

```bash
# Step 1 — Update packages and install dependencies
pkg update && pkg upgrade
pkg install nodejs git

# Step 2 — Clone the repository
git clone https://github.com/M41NUL/WA-Multi-Downloader.git
cd WA-Multi-Downloader

# Step 3 — Install Node.js dependencies
npm install

# Step 4 — Run the bot
npm start
```

---

### 🖥️ Windows / Linux

**Step 1 — Install Node.js & Git**
- **Windows:** Download [Node.js LTS](https://nodejs.org) and [Git](https://git-scm.com)
- **Linux:** `sudo apt install nodejs npm git`

```bash
# Step 2 — Clone the repository
git clone https://github.com/M41NUL/WA-Multi-Downloader.git
cd WA-Multi-Downloader

# Step 3 — Install dependencies
npm install

# Step 4 — Run the bot
npm start
```

> ***The `session/` folder will be created automatically to store authentication. Always follow the pairing code instructions shown in the terminal.***

---

### ⚡ Quick Start *(if Node.js v20+ and Git are already installed)*

```bash
git clone https://github.com/M41NUL/WA-Multi-Downloader.git
cd WA-Multi-Downloader
npm install
npm start
```

> ***💡 Follow the Pairing Code that appears in the terminal to connect your WhatsApp.***
